Projet DeefyApp
Fait par Justin EDOUARD et Alexis COUTURIER
Le site est sur le port 8888
La base de données est sur le port 8000
Pour pouvoir faire fonctionner projet il faut executer le fichier DONNEES.txt dans la base de données deefy
Fonctionnalité :
  -Inscription
  -Connexion
  -Déconnexion
  -Création de Playlist
  -Ajout de track a une Playlist
  -Lecture des tracks
